What is Meant by Probability Density Function (P.D.F)?

The probability function that illustrates the density of the continuous random variable which lies between a certain range of values is called the probability density function (pdf) in probability and statistics. Also, sometimes the probability distribution function or probability function is used to denote the probability density function(pdf).

However, the PDF is used to define the probability of the random variable falling within a certain range of values. Basically, this function is determined for continuous random variables, whereas the Probability mass function (PMF) is defined for discrete random variables.

FAQs on Free Online Mean of Probability density function calculator with steps

1. What does the probability density function (PDF) calculate?

The probability density function (PDF) allows you to find the probability of an event, like this:

  • For discrete distributions, the probability that X has values in an interval (a, b) is precisely the sum of the PDF of the possible discrete values of X in (a, b).
  • For continuous distributions, the probability that X has values in an interval (a, b) is exactly the area under its PDF in the interval (a, b).

2. What is the formula of probability density function?

The probability density function (pdf) f(x) of a continuous random variable X is defined as the derivative of the cdf F(x). Thus, the formula of PDF is f(x) = d/dx * F(x).
