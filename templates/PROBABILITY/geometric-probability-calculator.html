What is Geometric probability?

Geometric probability is used to describe the probability for a specified number of trails that takes place before the first success occurs. This probability is mainly built on the series of independent trails with two events. We will look into the formula that takes place to determine the geometric probability.

P(X=x) = (1-p)x-1* p

Where, x=1,2,3....

x = integer and there is no max values for x.

p = probability of success for single trail.

X = Random Variable.

Example Question for Geometric Probability

Example Question: If you flip a coin until it turns up the first tail. Find the geometric probability where it takes three flips to get the first tail?

Solution:

Given data is x=3, p=0.5

As we know the formula, P(X=x) = (1-p)x-1* p

Substitute the values in the given formula and the equation becomes,

P(X=3) = (1-0.5)2 *(0.5)

P(X=3) = (0.25)(0.5) = 0.125

Thus, the probability of three flips to get the first tail is 0.125.
