What is Discrete Probability Distribution?

The probability distribution is the possibility of each outcome of a random experiment. The two types of the probability distribution that are used for different purposes are Normal or Cumulative Probability Distribution and Binomial or Discrete Probability Distribution.

A discrete probability distribution is a distribution, where the set of outcomes is should be in discrete nature. An example is when a dice is rolled, then all the possible outcomes are discrete. In binomial distribution, the possible number of outcomes must be 2.

Discrete Probability Distribution Formulas

A discrete probability distribution takes the whole numbers from 0. The probability of a discrete random variable is the probability function, which is represented as f(x). Two important requirements for probability function are listed here:

  • f(x) > 0
  • ∑f(x) = 1

A discrete probability distribution can be represented in a couple of different ways. One is using a table and the second is using a graph. The third way is to use a formula for the probability function.

  • Discrete Uniform Probability Function is f(x) = 1/n. Where n is the number of values of x.

Discrete random variables can be explained using the expected value and variance.

  • Expected Value formula is E(x) = ∑x . f(x)
  • Variance formula is Var(x) = ∑(x - μ)² . f(x)
  • Standard deviation is SD = √[∑(x - μ)² . f(x)]

The binomial probability distribution is related to the binomial experiment. Every binomial experiment has a sequence of n trials with two possible outcomes in each trial. The outcomes are "success" and "failure" with probabilities of p and q = 1-p. The probability of x success in n trials is given by the binomial probability function.

  • Expected value E(x) = np
  • Variance Var(x) = npq = np(1-p)
  • Binomial Probability Function f(x) = nCx px qn-x

The Poisson probability distribution is helpful when the random variable measures the number of occurrences over an interval of time or space. Probabilities for a Poisson probability distribution can be find using the Poisson probability function.

  • Poisson Probability Function formula is f(x) = [μx e-μ]/x!

The hypergeometric probability distribution is similar to the binomial probability distribution. In a hypergeometric probability distribution, the number of success is r, N is the population.

  • Expected value formula is E(x) = n(r/N)
  • Variance formula is var(x) = n(N/r) (1 - r/N) (N-n/N-1)
  • Hypergeometric probabiity function is f(x) = [rCx N-rCn-x]/NCn
