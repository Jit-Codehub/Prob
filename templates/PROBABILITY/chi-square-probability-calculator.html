What is Chi Square Test?

Chi Square Test is a simple test that measures how observed values differ from the expected ones. You can use this to test how grades differ from intended distribution.

How to Calculate Chi Square?

You can find the Chi Square Value by using the formula χ2 = (observed value - expected value)² / expected value

You can use the chi square tables to compare one value with other values. There are two different factors in the table

  • Degrees of Freedom: We can get the Degree of Freedom by simply subtracting number of levels of analyzed variable minus one. For example, if you are given a grading example in which you have 5 grades the degree of freedom would become 4.
  • Significance Level: It id denoted using the symbol χ2. It is nothing but the probability of rejecting your hypothesis when it is actually true. The greater the risk is more extreme data is accepted as still valid.
