What is Uniform Probability Distribution?

In mathematics, the uniform probability distribution is the type of probability distribution where all the event outcomes are equal. In this, we have two types of probability distributions, they are discrete uniform distribution and continuous probability Distribution. For example, if you toss a coin it will be either head or tail, they both have the same chances.

F(x) = 1/(b-a), if (a <= x <= b)

F(x) = 0, if a > b.

Where,

a, b are integers.

How to Find Probability of Uniform Distribution?

Example:

Question 1: Find the probability of uniform distribution of a= 10, b=13?

Solution:

Given a= 10, b=13.

Here formula for a uniform probability distribution is

F(x) = 1/(b-a), if (a <= x <= b)

F(x) = 0, if a > b.

In this problem, a is greater than b, so we calculate 1/(b-a).

= 1/(13-10) = 1/3 = 0.33

Therefore, the uniform probability distribution is 0.33
